Leviblodgett.com

This is the first website I have ever done, and I am quite proud of it. I used twitter bootstrap for the base to make turning it into a responsive website easier, and to get used to the technology. This site was very customized from start to finish, I made each button in a very interactable way and was able to achieve my entire plan for the website that I had made in the beginning. The technologies of the command line, live-server, bower, NPM, Git, and Koala were all used as supporting developmental technologies when I was making the site. They expediated the entire process while making backups very often through the combination of the tech.

I have achieved everything I wanted to with this site, it is simple and displays a lot of information while giving the user the resources needed to learn about me or contact me with professional interest.

The following is the list of the technology that I used for the making of this website:

  • HTML 5 - Markup language
  • CSS - Style sheet language
  • SASS - Style sheet language
  • Twitter Bootstrap - Front-end framework
  • Koala - GUI/Compiler application for SASS
  • Git - Version control system (VCI)
  • Command Line - Linux & Windows
  • Font-awesome - Icon and CSS framework
  • Javascript - Diverse and powerful language
  • Bower - Package manager
  • NPM - Package manager for JS
  • Live-server - NPM/Bower extension

Jessicablodgett.com

This is a completely custom website I designed and made for my wife while she's in university. I didn't use bootstrap at all for a framework, just as a resource for some of the trickier button designs and such. I wanted to make this website responsive on my own terms so it would be completely evident I am comfortable with the technologies I'm using here, including responsive design which both of my sites have. I like to incorporate as much as what I know into the websites I make to make the process faster and the result better than ever.

The only thing that I wished I could've done with this site but was unable was making a PDF responsive, but that goes against the very nature of a PDF so I tried making it the best size possible for all screens.

  • Koala - GUI application for SASS
